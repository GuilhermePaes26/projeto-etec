HTML, que significa Hypertext Markup Language (Linguagem de Marcação de Hipertexto), é a linguagem padrão usada para criar e estruturar páginas da web.

O que é HTML?

Linguagem de Marcação: HTML é uma linguagem de marcação que usa tags para definir e estruturar o conteúdo em um documento web. Essas tags indicam ao navegador como exibir o texto, imagens, links e outros elementos.

Estrutura de Documento: HTML fornece a estrutura básica de um documento web, incluindo a definição de títulos, parágrafos, listas, links, imagens, tabelas e muito mais.

Semântica: HTML também permite o uso de tags semânticas que descrevem a função do conteúdo, como header, footer, article, e section. Isso ajuda a criar uma estrutura mais compreensível e acessível para os usuários e motores de busca.

Como Funciona o HTML?

Estabelecimento da Conexão Segura: Porta de Controle: FTPS geralmente usa a mesma porta de controle (porta 21) do FTP, mas a comunicação é criptografada após o estabelecimento da conexão. Portas de Dados: As portas usadas para a transferência de dados podem variar e dependem do modo de operação (ativo ou passivo).

Modos de Conexão: Modo Ativo: O cliente abre uma porta para receber dados e o servidor se conecta a essa porta. No FTPS ativo, a comunicação de dados é criptografada. Modo Passivo: O servidor abre uma porta para a comunicação de dados e o cliente se conecta a essa porta. No FTPS passivo, a porta de dados também é criptografada.

Características do HTML

  • Linguagem de Marcação: HTML é uma linguagem de marcação usada para criar e estruturar conteúdo para a web. Ela utiliza "tags" para identificar diferentes tipos de conteúdo e definir como eles devem ser exibidos pelos navegadores.
  • Estrutura Hierárquica: O HTML organiza o conteúdo de uma página da web em uma estrutura hierárquica. Isso significa que os elementos podem ser aninhados dentro de outros elementos, criando uma organização lógica e bem definida.
  • Elementos e Tags: HTML usa elementos, que são as unidades básicas de construção de uma página web. Cada elemento é representado por uma "tag" que define seu propósito, como criar títulos, parágrafos, listas, e links.
  • Atributos: Elementos HTML podem ter atributos que fornecem informações adicionais sobre o elemento. Esses atributos ajudam a definir aspectos como endereços de links, textos alternativos para imagens e estilos de apresentação.
  • Semântica: HTML5 introduziu elementos semânticos que melhoram a estrutura e a compreensão do conteúdo. Estes elementos têm significados específicos e ajudam a descrever o papel do conteúdo na página, como cabeçalhos, rodapés e seções.
  • Documentos Baseados em Texto: Os documentos HTML são arquivos de texto simples. Eles podem ser criados e editados com qualquer editor de texto e são salvos com a extensão .html ou .htm.
  • Compatibilidade com Navegadores: HTML é suportado por todos os navegadores web modernos. Isso significa que páginas criadas com HTML podem ser visualizadas em qualquer navegador, proporcionando consistência na apresentação do conteúdo.
  • Responsividade: Embora o HTML por si só não defina como a página deve se ajustar a diferentes tamanhos de tela, ele pode ser combinado com CSS para criar layouts responsivos que se adaptam a dispositivos móveis, tablets e desktops.
  • Integração com CSS e JavaScript: HTML pode ser usado em conjunto com CSS para estilizar o conteúdo e com JavaScript para adicionar interatividade. CSS controla a aparência, como cores e fontes, enquanto JavaScript permite funcionalidades dinâmicas.
  • Hierarquia e Navegação: HTML permite a criação de uma hierarquia clara e uma estrutura de navegação, facilitando a organização e a acessibilidade do conteúdo. Isso inclui a criação de menus, links e seções distintas.
  • Multimídia e Formulários: HTML permite a incorporação de elementos multimídia, como imagens e vídeos, e a criação de formulários para coleta de dados do usuário, como informações de contato e pesquisas.
  • Extensibilidade: HTML pode ser estendido e adaptado com a inclusão de novas tags e atributos conforme as necessidades da web evoluem. HTML5, por exemplo, trouxe novos elementos e atributos para suportar novas funcionalidades e padrões web.
