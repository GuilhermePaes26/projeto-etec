DNS (Domain Name System)

O DNS (Domain Name System) é um sistema fundamental para a operação da Internet. Ele mapeia nomes de domínios amigáveis, que são fáceis de lembrar, para endereços IP numéricos, que são usados por computadores para localizar e se comunicar com outros dispositivos na rede. O DNS funciona através de uma hierarquia de servidores que respondem a consultas de nomes de domínio e fornecem o endereço IP correspondente. Além de traduzir nomes de domínio, o DNS também gerencia registros de recursos como MX (Mail Exchange) para e-mail e CNAME (Canonical Name) para aliases de domínio.

Domínio

Um domínio é um identificador exclusivo que permite que um site seja acessado na Internet. A estrutura de um nome de domínio é composta por várias partes, incluindo:

Subdomínio: Parte opcional antes do nome principal (ex: "www" em "www.exemplo.com").

Nome do Domínio: A parte central do domínio que identifica o site (ex: "exemplo").

Extensão de Domínio: A parte final do domínio que especifica o tipo ou a localização geográfica (ex: ".com", ".org", ".net", ".edu", ".br").

O registro de um domínio garante que o nome desejado está disponível e permite que você controle como ele é utilizado. É essencial para criar uma identidade online única e profissional para indivíduos e empresas.

Empresas de hospedagem

Armazenamento de Dados: Os provedores de hospedagem oferecem servidores (computadores poderosos) onde os arquivos do site, como HTML, CSS, imagens e scripts, são armazenados. Esses servidores estão conectados à Internet, permitindo que qualquer pessoa com acesso à web possa visualizar o site.

Domínio e Acesso: O serviço de hospedagem permite que o site seja acessível através de um nome de domínio. Quando alguém digita o endereço do site no navegador, o provedor de hospedagem serve os arquivos do site ao visitante.

Representação: Endereços IPv6 são representados em notação hexadecimal, divididos por dois pontos, por exemplo, 2001:0db8:85a3:0000:0000:8a2e:0370:7334.
